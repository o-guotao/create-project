## 特性
- 可靠的代码兼容: 使用TS构建，提供完善的类型定义文件
- 极简的代码结构: 模块化设计，每个Hooks都单独一个文件，无多余的代码
- 完整的测试用例: 覆盖了所有Hooks的代码，保证每个Hooks的代码质量
- 完整的文档: 详细描述了每个Hooks的用法和参数，以及返回值和异常情况

## 安装
```bash
npm install gthooks
```
